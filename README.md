# Espacio VIP App | In Progress | 2025

Espacio VIP App is a desktop application developed to manage basic accounting workflows for a small carpentry business.

The application was created to replace manual Excel-based processes that were becoming inefficient and error-prone, providing a clearer and more structured way to track invoices, payments and client information.

This project was built and used internally to simplify daily operations.

---

##  Purpose

The main goal of this application is to:
- Organize invoices and payments
- Track partial and full payments from clients
- Reduce manual calculations and repetitive tasks
- Improve clarity and control over financial data

The focus was usability and practicality, not complex accounting automation.

---

##  Features

###  Invoice Management
- Register invoices with and without VAT
- Store invoice dates, amounts and client information
- Clear separation between sales and purchases

###  Payment Tracking
- Register multiple payments for a single invoice
- Track partial payments over time
- Automatically calculate remaining balances

###  Client & Provider Records
- Store basic client and provider information
- Filter invoices and payments by client
- Centralized view of all related transactions

###  Basic Summaries
- View totals and balances at a glance
- Reduce manual summation and reconciliation work

---

The application uses a local database to ensure simplicity and offline usage.
